# 바이브 코딩 테스트

간단한 숫자 맞추기 게임이 포함되어 있습니다.

사용법:

- 로컬에서 실행:

```bash
python3 guess_game.py
```

- 자동 검증(비대화형):

```bash
python3 guess_game.py --self-test
```

게임 동작:

- 컴퓨터가 1부터 100 사이의 숫자 하나를 무작위로 선택합니다.
- 플레이어는 숫자를 입력하여 정답을 맞추면 됩니다.
- 입력이 정수가 아니면 재입력을 요청합니다.

파일:

- `guess_game.py`: 메인 게임 스크립트

즐겁게 플레이하세요!
